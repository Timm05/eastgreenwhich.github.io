# East Greenwich Data Center — Community Campaign Website

A clean, responsive one-page advocacy site to oppose a proposed data center in
East Greenwich. Built with plain HTML and CSS

## Files
- `index.html` — page content and structure
- `styles.css` — all styling (colors, layout, responsive design)
- 
## Sections included
- **Hero** — headline + call to action
- **Why** — six reasons the data center shouldn't be built
- **How to Help** — four action steps + social links
- **About** — who the campaign is
- **Footer** — navigation + Instagram, Facebook, X links

## How to preview
Double-click `index.html`,
then visit http://localhost:8000

## Things to customize before going live
1. **Social links** — In `index.html`, replace every `href="#"` on the
   Instagram, Facebook, and X icons with your real profile URLs.

## How to publish it free
- **Netlify Drop:** go to [app.netlify.com/drop](https://app.netlify.com/drop)
  and drag this whole folder in. Done — you get a live URL in seconds.
- **GitHub Pages:** push the folder to a repo and enable Pages.
- **Cloudflare Pages / Vercel:** also free and drag-and-drop friendly.
